## 🎯 Project Overview
Comprehensive Security Operations Center (SOC) homelab simulating 
enterprise Active Directory environment with full threat detection 
and incident response capabilities.

## 🏗️ Architecture
[In Process of Making.]

## 🛠️ Technology Stack
- **Firewall/IDS:** pfSense Community Edition + Suricata
- **SIEM:** Wazuh 4.7.5
- **Domain Controller:** Windows Server 2019 + Active Directory
- **Endpoint Monitoring:** Sysmon (SwiftOnSecurity config)
- **Attack Platform:** Kali Linux 2024

## 🎓 Skills Demonstrated
- Active Directory administration & security
- SIEM rule creation and log analysis
- Network intrusion detection (Suricata)
- Endpoint detection and response (Sysmon + Wazuh)
- Attack simulation (Mimikatz, BloodHound, Impacket)
- Incident response and forensics
